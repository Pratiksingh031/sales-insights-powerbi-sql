
Sales Insights Dashboard (Power BI + SQL)
Project Overview

This project analyzes sales data across multiple Indian markets using SQL and Power BI to generate actionable business insights on revenue, sales quantity, and profitability. The dashboard enables stakeholders to monitor market performance, customer contribution, product trends, and profit margins through interactive visualizations.

Tools & Technologies
SQL (data extraction and analysis)
Microsoft Power BI (data visualization and dashboarding)
Data Modeling (relationships between transactions, customers, and date tables)
Key Insights

Sales Overview

Total Revenue: ₹985M
Total Sales Quantity: 2M units
Delhi NCR contributes over 50% of total revenue, indicating high market dependency
Revenue shows a declining trend towards 2020
High dependency on top customers, with one contributing ~₹413M
Data quality issues identified (e.g., blank product category)

Sales Performance

Revenue (2020): ₹142M
Sales Quantity: 350K units
Profit Margin: ~1.4%
Significant variation in profitability across markets
Certain regions show negative contribution, indicating losses
Monthly trends highlight possible external or seasonal impacts

Profit Analysis

Total Profit Margin: ₹24.7M
Delhi NCR is the strongest performing market in both revenue and profit
Some regions show negative margins, indicating operational inefficiencies
Profitability varies significantly across markets
SQL Analysis
Extracted and processed transaction and customer data
Performed market-wise and time-based analysis
Calculated revenue using joins with date tables
Filtered and analyzed multi-currency transactions
Business Insights
Revenue is concentrated in a few markets and customers
Overall profit margins are low despite high revenue
Several regions operate at low or negative profitability
Data inconsistencies impact analysis quality
Declining revenue trend suggests need for strategic improvements
How to Use
Clone the repository
Import the SQL file into a database
Open the .pbix file in Power BI Desktop
Update data source connections if required
Use filters to explore insights
